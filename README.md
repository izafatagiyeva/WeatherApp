# WeatherApp
A dynamic, real-time weather forecasting web application that delivers hyper-local updates based on the user's current location, alongside a global search feature to check weather conditions for any city worldwide. Built entirely with vanilla web technologies.

## Features 🚀
* **Geolocation Detection:** Automatically requests permission to fetch and display the user's local weather instantly upon page load.
* **Global City Search:** An intuitive search bar that allows users to type and search for live weather data in any city globally.
* **Detailed Weather Metrics:** Displays comprehensive atmospheric data including humidity, wind speed, pressure, "feels like" temperature, and weather descriptions.
* **Unit Toggle (Celsius / Fahrenheit):** Seamlessly switch between Metric (°C) and Imperial (°F) systems with a single click, updating all UI values dynamically.

## Tech Stack 🛠️
* **HTML5:** Semantic layouts to organize weather grids, widgets, and search controls.
* **CSS3:** Custom UI styling, responsive design with Flexbox/Grid, and dynamic visual transitions.
* **JavaScript (ES6+):** Complete handling of asynchronous API calls (`fetch`), Geolocation API integration, state management for unit conversion, and DOM manipulation.
* **External API:** [OpenWeatherMap API](https://openweathermap.org/) or serving accurate global meteorological data.
